管理培训生(产品方向)

工作地点：深圳市工作经验：2年学历：本科及以上
工作类型：全职招聘人数：2人发布时间：2016-8-15

职位描述

1、协助执行公司的产品开发策略，完成公司产品的规划、设计及上线工作。

2、协助渠道拓展，寻找优质资产，展开机构合作，对资产、项目和产品进行独立判断、筛选并推动引入。

3、协助参与产品规划、设计及开发工作，并独立完成产品的结构设计、包装，协助相关部门进行产品的营销 推广。

4、协助协调各板块资源，跟踪产品信息。定期及不定期开展与公司各板块的沟通，及时了解各板块以及销售 一线的产品需求，并跟踪在售产品销售情况。

5、参与产品的存续管理。根据部门的工作要求，参与执行产品存续管理工作。

任职要求

1、211、985院校（或海外知名大学）经济类、金融类或其它相关专业本科及以上学历；

2、学习和理解能力强，思路清晰，善于思考
