关于招聘

2017“牛犊”校园招聘面向那些学生？
主要对象是2017年毕业的本科生、硕士生、博士生的在校学生，同时也招聘2016年 -2017年期间毕业的海外留学生。
小牛招聘主要是招什么专业的学生，对专业有要求吗？
同学们可从我们校招官网发布的岗位介绍中了解具体岗位的专业要求，同时只要你的兴趣、能力与岗位需求相符，我们也欢迎跨专业的同学应聘。
本次招聘的主要工作地？
大部分在深圳，极少个在北京。
本次只录取宣讲覆盖的高校或地区的学生吗？
非宣讲覆盖的高校或地区的学生只有符合我们的岗位要求，参与到我们某个站点综合面试的，笔试和专业面试我们可以进行线上安排，欢迎全国各高校的同学们及海外留学生应聘。
小牛的选才标准是什么？选才过程中最看重的能力是什么？
同学们可从我们校招官网发布的岗位介绍中了解具体岗位的任职要求，不同的岗位关键任职能力有所不同，同时也希望更多懂得团队协作、有较强学习能力的小伙伴加入我们。
