小牛分期

深圳市小牛消费服务有限公司成立于2015年9月30日，注册资金人民币一亿元，是小牛资本管理集团旗下的全资子公司，是集团重要的战略业务板块。

小牛分期依托集团雄厚的金融背景，承载集团“互联网化”基因，以“极致的用户体验”为宗旨，通过引进国际化的管理团队以及精细化的金融风控体系，致力于打造创新型综合消费金融服务平台，通过科技化、场景化的金融服务为用户提供舒适、便捷的体验，满足广大消费者和商户的个性化需求，为消费金融业注入新的活力。

公司将稳步推进全国性战略布局，并通过加强国内外合作，加大创新力度，借鉴先进国际经验及增强综合竞争能力等新途径，成为全球消费金融行业的建设者和引领者，推动产业发展升级，增强消费对经济的拉动力，不断为我国消费金融改革创新做出自己应有的贡献。
