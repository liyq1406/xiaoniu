小牛价值观

事实上，小牛的用户群，既包括高净值人群、也包括普通人、甚至是穷人
过去有句流行的话叫，“为富人说话，为穷人办事”
尽量让两者的收益相近、风控相等

小牛说，它要成为
国际化综合金融服务集团的民族范本
这不是星辰大海，这不是遥不可及...
金融的所谓体量，从来不是价值的唯一衡量者
最重要的范本意义：小牛真正辨认、完善和
坚守它的使命、它的伦理、它的价值...
像伟大诗人里尔克写的那样
“我辨认出风暴而激动如大海”

换句话说，小牛是谁，做成怎样
它的前提应该是——
小牛可以给这个社会带来什么？
这就是所谓的初心哲学：
你出发的目的，决定了你可以走完的距离
