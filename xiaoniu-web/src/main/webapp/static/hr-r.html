公司介绍

小牛资本管理集团以互联网金融、财富管理和投资管理为主要业务的创新型金融服务机构，在北京、上海、深圳设有三大管理中心，分支机构遍布全国。

集团视人才为企业核心资本，重视人才素质提升，培养德才兼备的专业化人才，努力打造学习型团队。同时，发展成果惠及员工，关爱员工，使员工与企业共成长。

秉承以心为本，公平正义，兼容创新，卓越共成的核心理念，为卓越而生，引领市场金融，成为全球一流金融服务企业，让人人平等享有金融服务。

截至目前，在北京、上海、深圳、香港设立四大管理中心，全国27个省市150多个城市开设了300多家分支机构，境外业务辐射迪拜、澳洲、伦敦、纽约等国际金融中心

核心竞争力

我们将服务实体经济、服务人们生活视为我们存在的价值。金融是跨时间、跨空间的价值交换，它的本质是实现资源的优化配置，增强国力，发展民生，服务生活。我们将持续创新视为我们发展的动力，我们关注社会中需要变革的领域...

  • 潜力巨大的服务市场

  • 优质稳定的资产端

  • 合理的资金成本

  • 投融闭环

  • 强大的数据支撑

社会责任

普惠人群数：截至目前，我们服务的客户数逾300万，创造了2万个就业岗位，辐射了上千万的就业机会。

投资公平：我们的客户包括高净值人群，也包括普通人，甚至穷人。我们尽量让两者的收益相近、风控相等。

绿色金融：环保、节能、清洁能源、清洁交通等绿色产业项目是我们关注的投资主题。绿色金融不仅是国家战略，更是企业的社会责任和公民良心。
