薪酬福利

现金回报

我们为员工提供富有市场竞争力的薪酬待遇，每年都会对员工的薪酬进行回顾。每个季度我们还会根据公司的整体业绩和员工绩效提供季度奖金，体现公司与员工同成长、共分享的理念。

员工保障

我们为员工提供完善的保障计划，除了国家规定的养老保险、医疗保险、工伤保险、失业保险，生育保险、住房公积金之外，我们还为员工提供每年一次的免费健康体检。我们还会为优秀的员工免费提供商业保险。

带薪休假

除了法定的各种假期，我们还人性化地提供每月一天的带薪病假。

其它福利

公司竭尽所能为员工提供各种实用的福利，营造舒适的工作环境，每天有水果和下午茶提供。为了丰富员工的业余生活，公司成立了各种体育协会，定期组织公司旅游，举办生日会……
