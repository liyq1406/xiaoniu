小牛新财富

小牛新财富为高净值人士及高成长性人群提供全球资产配置服务和全面的理财顾问服务。涉及领域涵盖私募基金、境外房产经纪、家族信托、留学移民、海外医疗、海外养老。截至目前， 我们已成为中国证券投资基金业协会备案的私募基金管理人，并取得基金销售牌照和融资租赁牌照。

发展历程

2013年5月9日成立；

2013年:

5月-被深圳市私募基金协会评选为理事会理事单位

7月-凤凰卫视《大政商道》节目组采访小牛资本集团相关领导

8月-2013前海金融论道暨深港金融创新盛典，荣获“最佳产品创新奖”

12月-董事长彭铁应邀出席2013年中国创新资本年会

2014年:

3月-董事长彭铁应邀出席由万科集团董事局主席王石先生发起 深商会携手剑桥大学商学院举办的“深商剑桥论坛”

5月-成为广东省互联网金融协会理事会员单位

6月-小牛投资正式成为中国证券投资基金业协会备案的私募基金管理人

8月-荣获金贝奖“最具发展潜力互联网金融机构”

10月-获得由中国社科院首发的A级评级称号

2015年:

1月-小牛入选2015中国自主品牌百佳企业

4月-董事长彭铁随同唐杰副市长、王石等60多名深圳企业家来到美国西雅图考察

9月-凤凰生活周刊对小牛新财富管理有限公司CEO刘金科进行专访

10月-小牛资本“Let’s Niubility”广告亮相美国纽约时代广场向祖国母亲66华诞献礼

12月-小牛新财富获得“2015年度中国卓越综合财富管理公司”称号
